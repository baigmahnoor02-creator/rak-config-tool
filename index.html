How to use this tool

  1. Connect: power on the node, tap Connect to device, and pick RAK4631-IIOT.
  2. Unlock: enter the PIN (config PIN gives full access; read-only PIN shows diagnostics only) and tap Unlock.
  3. Live readout: readings refresh automatically every few seconds (network, battery, pressure, raw ADC, GPS, motion).
  4. 4-20 mA calibration: in Sensor, set type 4-20mA. At 0 psi tap Fetch on the first row (zero). At a known/full-scale pressure tap Fetch on the second row and set the full-scale value, then Apply calibration.
  5. Intervals & actions: adjust sample/uplink/GPS timing; use Force uplink or Install mode for commissioning.
  6. LoRaWAN: enter OTAA keys (DevEUI, AppEUI, AppKey), set Class/ADR, write keys, then reboot to join.
  7. Backup: tap Read current config to review settings (the AppKey is never exported).

Tip: use Android Chrome over https. Keep Bluetooth on. Tap Help again to close.

Movement sensitivity

For slow equipment, GPS movement is the reliable signal; the accelerometer flags start-up. A higher minimum-satellites value gives a more reliable position fix (it may take a little longer to lock).

Sensor

4-20 mA calibration (2-point)

Standard current loop: 4 mA = zero, 20 mA = full-scale. Connect the sensor, then use Fetch to capture the live raw value at each point.

At 0 psi (4 mA), tap Fetch on the first row. At a known pressure (or full scale), tap Fetch on the second row and set the full-scale value.
